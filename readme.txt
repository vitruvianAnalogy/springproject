How to create a Spring project.
1. Choose a maven project from eclipse. No need to choose archetypes.
2. Find the correct spring framework archetype on the web.
3. Use the group id, artifact id and version and put it in the pom.xml
4. Create a spring.xml file for spring configuration. Usually put under src/main/resources folder
5. Start using the configuration in code.


Use application context for accessing beans from the config.
